# Sahil Otavanekar - Data Science & AI Portfolio

Welcome to my personal portfolio repository! This website is intentionally designed and built to showcase my projects, skills, and technical experience in Data Science, Machine Learning, and AI Development to recruiters and hiring managers.

## 🌟 Overview

This repository contains the source code for an interactive resume and project gallery, engineered with a modern and responsive technology stack:
- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4, Framer Motion for smooth animations
- **Icons**: Lucide React

## 🚀 Featured Projects

The portfolio highlights a diverse range of projects demonstrating end-to-end data and AI engineering capabilities:

1. **AI-Powered Telegram Lead Generator**
   - *Tech Stack*: Python, Gemini AI, Playwright, Airtable, Serverless Modal
   - *Description*: Automates lead generation by scraping Google Maps via natural language commands using Gemini 2.0.

2. **Multimodal AI Medical Chatbot**
   - *Tech Stack*: Next.js, FastAPI, Gemini 2.5 Flash, Supabase
   - *Description*: An AI-powered medical assistant capable of interpreting symptoms using text, image, and voice capabilities, equipped with robust safety guardrails.

3. **Startup Funding Intelligence Platform**
   - *Tech Stack*: React, FastAPI, XGBoost, MLflow, Docker, Supabase
   - *Description*: A full-stack Business Intelligence tool predicting startup funding success using scalable ML models and real-time interactive analytics.

4. **Monte Carlo Portfolio Risk Assessment**
   - *Tech Stack*: Python, Streamlit, Pandas
   - *Description*: Simulates financial portfolios using historical volatility to forecast possible future performance and potential risk using stochastic modeling.

5. **Customer Churn Analysis Dashboard**
   - *Tech Stack*: Python, Streamlit, Plotly, Scikit-Learn
   - *Description*: Interactive dashboard analyzing predictive factors driving customer churn to identify high-risk segments using targeted Machine Learning.

6. **Fashion Product Recommendation**
   - *Tech Stack*: Python, Deep Learning, CNNs (ResNet50), Streamlit
   - *Description*: Visual similarity-based recommendation engine for fashion items using deep learning feature extraction and fast KNN lookups.

## 💻 Running the Portfolio Locally

If you'd like to inspect the code and run the portfolio website on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SahilOtavanekar/Portfolio-Website.git
   cd Portfolio-Website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the website:** Open the provided local URL (typically `http://localhost:5173`) in your web browser.
